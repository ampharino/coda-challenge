# coda-challenge

## How to run
This project was developed using node18. Recommend using [Volta](https://volta.sh/) to manage node versions.

```sh
npm install
npm run dev
```
Navigate to `localhost:5173`

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```
